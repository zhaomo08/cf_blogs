## Welcome to My Blog

This is my first official blog post! I finally have my own blogging space.

### Why Blog?

Blogging has many benefits:

1. **Document Life**: Record daily observations and thoughts
2. **Share Knowledge**: Share what I've learned with others
3. **Improve Writing**: Enhance expression through continuous writing
4. **Make Friends**: Meet like-minded people through blogging

### My Blog Tech Stack

This blog uses the following technologies:

- **Astro 5**: Modern static site generator
- **Cloudflare Pages**: Free static site hosting
- **Cloudflare R2**: Image storage
- **Pagefind**: Full-text search
- **Leaflet.js**: Map display

### Blog Features

✅ Bilingual support (Chinese/English)  
✅ Dark mode  
✅ Full-text search  
✅ RSS feed  
✅ Location maps  
✅ Responsive design
