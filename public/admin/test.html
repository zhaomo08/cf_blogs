🔧 快速设置

如果你还没有配置管理后台，请按以下步骤操作：

方法 1：使用设置脚本（推荐）

npm run setup-admin

方法 2：手动配置

  1. 编辑 public/admin/config.yml
  2. 将 YOUR_GITHUB_USERNAME/YOUR_REPO_NAME 改为你的仓库
  3. 创建 GitHub OAuth App
  4. 部署 OAuth Worker

详细说明请查看 README.md

🚀 访问管理后台

配置完成后，访问管理后台：

打开管理后台
