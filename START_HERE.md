# 🎉 欢迎使用你的博客管理后台！

## 🚀 立即开始（3 种方式）

### 方式 1：本地测试（最快，1 分钟）⚡

适合：想立即体验管理后台

```bash
# 终端 1：启动 Decap Server
npx decap-server

# 终端 2：启动开发服务器  
npm run dev
```

然后编辑 `public/admin/config.yml`，取消注释：
```yaml
local_backend: true
```

访问：http://localhost:4321/admin/ （无需登录）

---

### 方式 2：使用配置向导（推荐）⭐

适合：准备部署到生产环境

```bash
npm run setup-admin
```

按提示选择配置模式，脚本会自动更新配置文件。

---

### 方式 3：手动配置（完全控制）🔧

适合：想要完全了解配置过程

查看详细文档：
- [QUICK_START_ADMIN.md](./QUICK_START_ADMIN.md) - 5 分钟快速开始
- [ADMIN_SETUP.md](./ADMIN_SETUP.md) - 完整设置指南

---

## 📝 管理后台功能

访问 `/admin/` 后，你可以：

✍️ **可视化编辑**
- 所见即所得的 Markdown 编辑器
- 实时预览
- 语法高亮

📸 **图片管理**
- 直接拖拽上传
- 支持外部图片 URL
- 自动优化

🗺️ **地图功能**
- 可视化添加拍摄地
- 自动在文章底部显示地图

🏷️ **内容管理**
- 标签分类
- 草稿功能
- 发布日期设置

🌍 **多语言**
- 中文和英文独立管理
- 一键切换

http://localhost:4321/admin/test.html

http://localhost:4321/admin/

1. 选择"博客文章（中文）"
2. 点击"New 文章"
3. 填写标题、描述、日期
4. 写正文
5. 点击"Publish"

## 💡 使用建议

### 开发阶段
使用**本地模式**，快速迭代，无需认证。

### 生产环境
使用 **GitHub OAuth** 或 **Netlify Identity**，安全可靠。

### 团队协作
启用 `editorial_workflow`，实现草稿→审核→发布流程。
